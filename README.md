# 📊 Sales Analytics Dashboard

## 📌 Overview
An interactive Sales Analytics Dashboard built using Power BI, SQL, and Microsoft Excel to analyze customer behavior, product performance, and monthly sales trends.

## 🛠 Tools Used
- Power BI
- SQL
- Microsoft Excel

## 📈 Key KPIs
- Total Sales: 574K
- Total Customers: 269
- Total Transactions: 1K
- Average Order Value: 509.77
- Average Basket Size: 2.50

![Dashboard](./Screenshot%202026-07-21%20041403.png)

## 🔍 Key Insights
- Total Sales reached **574K**.
- Female customers contributed slightly more sales than male customers.
- Customers aged **31–40** generated the highest sales.
- Groceries was the highest-selling product category.
- Sales peaked during the middle of the year.
